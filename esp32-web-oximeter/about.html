ESP32 血氧心率监测仪

关于项目

本项目是基于 ESP32 开发的一款多功能血氧心率监测仪，采用非侵入式测量技术，可实时监测用户的心率和血氧饱和度。

技术特点

  • 采用 ESP32 高性能微控制器，支持 WiFi 连接
  • 使用 MAX30102 传感器进行生物信号采集
  • 通过 Chart.js 实现数据可视化，支持历史数据查看
  • 响应式设计，支持 PC 和移动端访问
  • 支持历史数据导出功能

使用说明

  1. 将手指轻轻放在血氧传感器上
  2. 等待 2-3 秒，系统将自动开始测量
  3. 在监测仪页面查看实时心率和血氧数据
  4. 可以在设置页面调整测量参数
  5. 通过导出按钮保存历史数据

注意事项

  • 测量时请保持手指稳定，避免晃动
  • 请勿在运动、进食或饮酒后立即测量
  • 本设备仅供参考，不能替代专业医疗设备
  • 若测量数据异常，请咨询专业医生
